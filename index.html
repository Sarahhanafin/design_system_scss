How to use this design system


Using a design system means following its rules and components to build consistent and easy-to-use interfaces. Here’s how to use it:


  1. Start with the Foundations – Use the system’s colours, fonts, spacing, and grids. Pick the right heading and body text sizes, use spacing tokens (4, 8, 16, 32, 64) to separate content, and choose a 6- or 12-column grid for layouts.


  2. Use components – Add buttons, input fields, checkboxes, toggles, and icons from the system. Use primary, secondary, or error buttons depending on the action. Add input fields with clear labels and placeholders, and use toggles or checkboxes where needed.


  3. Give feedback – Show success and error states when users complete actions, and use skeletons while content is loading.


  4. Create hierarchy – Use headings, body text, labels, captions, and placeholders consistently so content is easy to read. Add shadows or gradients on buttons to show importance


  5. Stay consistent – Keep icons, buttons, spacing, and interactions the same across your interface. This makes the design feel polished and easy to use.


  6. Check accessibility – Make sure colours, text sizes, and interactive elements are easy to see and use. Always pair icons with text and avoid relying on colour alone to convey meaning.


Using a design system this way saves time, keeps designs consistent, and makes your interface clear and user-friendly.
