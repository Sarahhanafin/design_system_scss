Spacing

The spacing system uses consistent steps of 4, 8, 16, 32, and 64 to keep layouts clean and organised. Spacing is used to control the layout and structure of a page by adding consistent margins and padding between elements. It helps improve readability, organization, and overall visual balance. In this design system, spacing is applied using predefined classes so everything stays consistent across a project. To use it, you can copy and paste the spacing classes into your HTML and apply them to elements where needed (for example, adding margin or padding).
