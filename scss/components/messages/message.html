Messages

Success, error, and warning messages are used to give users clear feedback after an action. Success messages confirm that something worked correctly, error messages show when something has gone wrong, and warning messages alert users to something they should check before continuing. In this design system, you can simply copy and paste the provided HTML into your project and apply the correct classes (like message--success, message--error, or message--warning). As long as you include the design system stylesheet, the messages will automatically be styled and display correctly without any extra setup.
