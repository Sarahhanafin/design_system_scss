Dropdown

Dropdown menus are used to let users choose one option from a list in a compact way. They are helpful when there are multiple choices but you want to save space, such as selecting a country or category. In this design system, you can simply copy and paste the provided HTML into your project and apply the correct classes for the dropdown. As long as you include the design system stylesheet, the dropdown will automatically be styled and function correctly.
