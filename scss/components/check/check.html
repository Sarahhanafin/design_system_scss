Checkboxes & Radio Buttons

Checkboxes and radio buttons are used to let users make choices in a form. Checkboxes allow users to select one or more options, while radio buttons let users choose only one option from a group. In this design system, you can simply copy and paste the HTML example you need into your project and apply the correct classes, such as control--checkbox or control--radio. You can also add size classes like control--sm, control--md, control--lg, or control--xl. For radio buttons, make sure all options in the same group share the same name attribute so only one can be selected. As long as the design system stylesheet is included, the components will automatically be styled and work correctly.
