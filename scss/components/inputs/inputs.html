Inputs

Input boxes are used to collect information like names, emails, dates, and numbers. Every input should have a label so users know what to type. You can also add icons to help show what the input is for, like a calendar for dates or an envelope for email. Inputs should clearly show different states, such as normal, disabled, or error, so users know if something is wrong or not available.
To use this design system, copy the HTML structure and use the same class names like input-group, input-wrapper, has-error, and is-disabled. Do not style inputs separately—use the shared SCSS styles instead. This keeps everything looking the same and makes it easier to build clean, consistent forms across your project.
