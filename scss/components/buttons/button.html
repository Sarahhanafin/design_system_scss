Buttons

Buttons are used to trigger actions, such as submitting a form, saving changes, or navigating to another page. In this design system, there are different types like primary, secondary, and error buttons to show importance or intent, as well as size options like small, medium, and large to fit different layouts. Buttons also include states (such as hover, active, or disabled) to give users clear feedback when interacting with them. To use them, you can copy and paste the provided HTML into your project and apply the appropriate classes (e.g. button--primary, button--secondary, button--error, along with size classes). As long as you include the design system stylesheet, the buttons will automatically be styled and behave correctly without any extra setup.
