Buttons

Buttons trigger actions and allow users to interact with the interface. They communicate what will happen when a user clicks or taps them. Buttons should always use clear, action-oriented labels and follow consistent styling across the product. Buttons are one of the most frequently used interactive elements in the interface and should be used to initiate immediate actions such as submitting forms, saving changes, navigating workflows, or confirming decisions.
